## Projeto de doações via PIX

Este repositório contém a aplicação web utilizada para receber doações para animais de rua por meio de pagamentos PIX.

### Desenvolvimento local

Requisitos:

- Node.js e npm

Passos:

```sh
npm install
npm run dev
```

### Tecnologias principais

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
